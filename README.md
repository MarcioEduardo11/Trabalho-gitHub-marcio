# Trabalho sobre o GitHub

São ferramentas que por meio delas eu consigo mandar meu projeto do VsCode para o GitHub para obter mais praticidade.

## Passo a passo
* crie uma pasta 
* abra ela com o VsCode
* Crie um novo arquivo
* Crie o README.md
* Abra o terminal
* De um "git version" para ver a versão
* De um git init
* De um git config --global user.name "Marcio"
* De um git config --global user.email "engs-marcioaraujo@camporeal.edu.br"
* De um git status
* De um git add .
* De git commit -m ""
